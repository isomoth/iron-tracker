# Iron Tracker

1.62 billion people suffer from anemia (iron deficiency). Nutrition is a decisive factor in many cases. This web application contains a database of ca. 8000 common foods that you can search for and filter through the two nutrients that help combat this condition: Iron and Vitamin C.
